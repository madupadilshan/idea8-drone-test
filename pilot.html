window.djiBridge is only injected by the Pilot 2 app, so a normal browser can't use it. To run it correctly:
  1. Host this file on your PC: in this folder run python -m http.server 8080 (or npx http-server -p 8080).
  2. On the controller, open Pilot 2 → Cloud Services and enter http://192.168.20.84:8080/pilot.html (your PC's IP).
  3. Pilot 2 opens it in its own web view, where the bridge exists, and this banner disappears.
